# 쇼핑몰 프로젝트 
> 의류, 잡화 등등 여러 카테고리의 상품을 쇼핑할 수 있는 웹 페이지
 * UI/UX 디자이너 1명, 프론트엔드 2명, 백엔드 1명이 팀을 이루어 프로젝트를 진행했고 저는 백엔드 파트를 담당했습니다. Aws ec2 서버에 배포했고 프록시 서버로 nginx를 사용했습니다.

## 프로젝트 링크
  [프로젝트 링크](https://orday.vercel.app/)

## 프로젝트 환경
  * Spring Boot 2.7.2
  * MySQL 8.0.33
  * Ubuntu 20.04
  * Nginx 1.18.0
  * Redis 5.0.7

## 배포
  * AWS EC2, S3, RDS, Route53
